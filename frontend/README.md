# Frontend (React + Vite + Tailwind + shadcn style)

## 启动

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 设计系统

- 视觉方向：现代金融专业风（浅色高对比 + 卡片化布局）
- 技术实现：Tailwind CSS + shadcn/ui 组件模式
- 关键目录：
  - `src/components/ui/` 基础 UI 组件
  - `src/components/layout/` 应用壳层
  - `src/pages/` 页面级视图

## 页面

- 总览
- 资产负债录入
- CSV 导入
- 分析看板
- 设置

## 图表

- 趋势图（Line）
- 波动率（Bar）
- 相关性矩阵（Heatmap）
- 桑基图（Sankey）
